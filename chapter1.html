Uses of Vectors

Position, Velocity, Acceleration are often all modeled as vectors. In p5.js we can use the p5.Vector class

  • Position: The movement from the origin to the current location
  • Velocity: The vector represents the change in position over time or frames
  • Acceleration: The vector for the change in velocity over time or frames

Reminder for Subtraction

In subtraction, to get a vector C pointing from A to B, start with the end point in the equation:C = B - A
